주요 특징

🎮

Phaser 3.85.0

강력하고 유연한 2D 게임 엔진으로 브라우저에서 바로 게임을 만들 수 있습니다

📘

TypeScript 5.x

타입 안전성으로 버그를 줄이고 더 안정적인 코드를 작성할 수 있습니다

⚡

Vite 5.x

초고속 개발 서버와 즉시 모듈 갱신(HMR)으로 개발 생산성을 극대화합니다

🏗️

Scene 기반

확장 가능한 아키텍처로 복잡한 게임도 쉽게 구조화할 수 있습니다

빠른 시작

의존성 설치

프로젝트 디렉토리에서 다음 명령어를 실행합니다:

pnpm install

💡 npm을 사용하는 경우 npm install을 사용하세요

개발 서버 실행

코드를 수정하면 자동으로 새로고침됩니다:

pnpm dev

브라우저에서 http://localhost:3000 으로 열립니다

프로덕션 빌드

최적화된 빌드를 생성하여 배포합니다:

pnpm build

dist/ 디렉토리에 빌드 파일이 생성됩니다

추가 명령어

코드 린팅
pnpm lint
코드 포맷팅
pnpm format
빌드 미리보기
pnpm preview

게임 데모

지금 플레이해보세요

오른쪽의 게임을 직접 플레이할 수 있습니다. 간단한 플랫폼 게임으로 스타터킷의 기본 기능을 보여줍니다.

컨트롤

  • 방향키 또는 WASD - 이동
  • 스페이스 또는 W - 점프
  • ESC - 메뉴로

게임 흐름

  • BootScene - 에셋 로딩
  • MainMenuScene - 메뉴 화면
  • GameScene - 메인 게임

⚙️ 설정 최소화

복잡한 설정 없이 바로 게임 개발에 집중하세요. TypeScript, Vite, ESLint가 미리 구성되어 있습니다.

📈 확장 가능한 구조

Scene 기반의 아키텍처로 게임이 복잡해져도 코드를 체계적으로 관리할 수 있습니다.

🚀 빠른 개발

Vite의 HMR로 코드를 수정하면 즉시 반영됩니다. 개발 중 브라우저 새로고침이 필요 없습니다.

📚 명확한 예제
